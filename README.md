# Onebox Cart 

Onebox Cart is an Microservice responsible for cart functions, such as creat a cart, add products to it, delete it and more functions like so.

## Technologies
- Java 11
- Springboot 2.7.9
- Lombok
- Oracle DB
- Maven
- Open API

## Installation

Use the package manager maven to install the dependencies.

```bash
mvn clean install 
```
